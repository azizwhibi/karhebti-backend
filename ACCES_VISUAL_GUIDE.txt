    Avant de commencer :

    ☐ Compte garage_owner créé (role = 'garage_owner')
    ☐ Breakdown de test dans la DB (ID = 1)
    ☐ App compilée et installée

    Pour tester :

    ☐ Login comme prop.garage@example.com
    ☐ Home Screen chargé
    ☐ Scroll vers le bas
    ☐ Trouvé section "🚨 SOS Management"
    ☐ Bouton rouge visible
    ☐ Click sur "🚨 Test SOS Details"
    ☐ Écran GarageBreakdownDetails s'ouvre
    ☐ Informations affichées correctement
    ☐ Bouton "Accepter" testé
    ☐ Bouton "Refuser" testé


┌──────────────────────────────────────────────────────────────────────────────┐
│                        🔧 CONFIGURATION RAPIDE                               │
└──────────────────────────────────────────────────────────────────────────────┘

    1. Vérifier le rôle de l'utilisateur :

       SELECT email, role FROM users
       WHERE email = 'prop.garage@example.com';

       → Doit afficher : role = 'garage_owner'

    2. Créer un breakdown de test :

       INSERT INTO breakdowns (userId, type, description, latitude, longitude, status)
       VALUES (1, 'PNEU', 'Test', 36.8065, 10.1815, 'PENDING');

    3. Compiler et installer :

       ./gradlew assembleDebug
       adb install -r app/build/outputs/apk/debug/app-debug.apk

       ✅ Solution : Vérifiez que role = 'garage_owner'
